# Классическое исчисление высказываний — конспект

Источник: `Классическое исчисление высказываний.pdf` (полный текст — в PDF).

---

## 2.1. Пропозициональная логика

- **Пропозициональные переменные** — места для простых высказываний (`p,q,r,…`).
- **Связки** — строят из формул новые формулы; значение сложной формулы определяется таблицей истинности по значениям частей (**истинно-функциональные** связки).
- **Логика высказываний** — язык: только пропозициональные переменные + связки.

---

## 2.2. Связки: смысл

| Связка | Кратко |
|--------|--------|
| `¬A` | истина ⇔ `A` ложно |
| `A ∧ B` | истина ⇔ оба истинны |
| `A ∨ B` | истина ⇔ хотя бы одно истинно (в лекции также различают «обычную» и **строгую** дизъюнкцию — «ровно одно»; строгую выражают через `∧`, `∨`, `→`) |
| `A → B` | ложь **только** при `A=1`, `B=0`; иначе истина. **Антецедент** / **консеквент**. |
| `A ↔ B` (или `≡`) | оба истинны или оба ложны |

**Функциональная полнота:** набор связок, через который выражается любая функция истинности; в курсе указан полный набор с `¬, ∧, ∨, →`.

---

## 2.3. Алфавит, формула, подформула

- Алфавит: переменные высказываний; связки (`&`, `∨`, `→`, `¬`); скобки.
- **Формула**: переменные; если `A,B` формулы — `(A*B)` и `¬A` (точные правила — в PDF).
- Нельзя опускать `&` и скобки, если меняется смысл.
- **Подформула** — «кусок» формулы по индуктивному правилу (см. примеры 2.1–2.2 в PDF).

---

## 2.4. Классы формул по таблице истинности

- **Общезначимая (тавтология)** — всегда 1.
- **Тождественно ложная** — всегда 0.
- **Выполнимая** — 1 хотя бы на одном наборе переменных.
- **Опровержимая** — 0 хотя бы на одном наборе.

Тавтология ⇒ выполнимая. Противоречие ⇒ опровержимая.

---

## 2.4 (продолжение). Аксиомы (11 штук, группы I–IV)

**I. Импликация**

- I1: `x → (y → x)`
- I2: `(x → (y → z)) → ((x → y) → (x → z))`

**II. Конъюнкция**

- II1: `x ∧ y → x`
- II2: `x ∧ y → y`
- II3: `(z → x) → ((z → y) → (z → x ∧ y))`

**III. Дизъюнкция**

- III1: `x → x ∨ y`
- III2: `y → x ∨ y`
- III3: `(x → z) → ((y → z) → (x ∨ y → z))`

**IV. Отрицание и контрапозиция**

- IV1: `(x → y) → (¬y → ¬x)`
- IV2: `¬¬x → x`
- IV3: `x → ¬¬x`

(В PDF — полная формулировка; здесь схемы в стиле курса.)

## 2.5. Правила вывода

1. **Подстановка:** если `⊢ A` и `x` — переменная, `B` — формула, то после подстановки `B` вместо `x` везде в `A` снова доказуемо (см. уточнения для `¬`, связок в PDF).
2. **Правило заключения (modus ponens, ПЗ):** из `⊢ B` и `⊢ B → C` следует `⊢ C`.

Запись: `⊢` — «доказуемо».

**Определение 2.15:** доказуемая формула — аксиома или получена из доказуемых подстановкой и ПЗ.

---

## Как пользоваться в задачах

- Доказать `⊢ F` — построить цепочку от аксиом с подстановками и ПЗ (см. примеры 2.6+ в PDF).
- Для **семантики** (истина/ложь) — таблица истинности без исчисления.
