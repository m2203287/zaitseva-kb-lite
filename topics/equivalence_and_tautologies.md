# Equivalence and Tautologies

## To prove equivalence
Show both formulas have same truth value for every interpretation.

## To disprove equivalence
Find one interpretation where values differ.

## Typical non-equivalence source
- Wrong quantifier movement across implication/disjunction.
- Confusing `exists x P(x) -> Q` with `forall x (P(x) -> Q)`.

## Useful reminders
- `forall x (A and B) <-> (forall x A) and (forall x B)`.
- `exists x (A or B) <-> (exists x A) or (exists x B)`.
- Be careful with mixed quantifiers and implication.

## Proof formatting
- State interpretation/domain explicitly.
- Evaluate each critical subformula.
- Conclude with one sentence: equivalent / not equivalent.
