# Predicate Logic Essentials

## Core objects
- Predicate variable: `P(x)`, `Q(x,y)`.
- Quantifiers: universal `forall`, existential `exists`.
- Closed formula: no free variables.

## Standard checks
- Satisfiable: true under at least one interpretation.
- Unsatisfiable: false under all interpretations.
- Tautology (for predicates): true under all interpretations.

## Frequent techniques
- Build a counterexample interpretation to disprove equivalence.
- Replace predicate symbols with concrete predicates over a chosen set.
- For one-element domain, simplify quantifier behavior carefully.
- For implications, separate cases when antecedent is false.

## Quick workflow
1. Identify formula type (satisfiable / tautology / equivalence).
2. Pick method: direct proof or counterexample.
3. Write interpretation and evaluate subformulas.
4. State final verdict clearly.
